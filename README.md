# DemExamen_8 Модуль№4_8 Куницын Кирилл
Версия проекта: 1.0
Название: "DemExamen_8"
О проекте: Данный проект создан для авторизации пользователей и администратора. Вход проводится через окно авторизации, при 3 неправильных ввода пароля или капчи, пользователь будет заблокирован, а разблокировать сможет только администратор. Администратор заблокирован быть не может.
